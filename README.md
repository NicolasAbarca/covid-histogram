# Covid Histogram

web-application that shows histogram with drilldown visualization.

## Installation

In the project directory, you can run:

```bash
# If you use npm:
npm install

# Or if you use Yarn:
yarn
```

## running the App

In the project directory, you can run:

```bash
# If you use npm:
npm start

# Or if you use Yarn:
yarn start
```
